# rock-paper-scissors-python
A simple Python console-based Rock, Paper, Scissors game that uses ASCII art to display each choice.
The player selects Rock, Paper, or Scissors, the computer generates a random choice, and the game determines the winner.

Features:

User input for selecting Rock, Paper, or Scissors

Random computer choice

ASCII art representation for each gesture

Win / lose / draw logic

Basic invalid input handling

Perfect as a beginner-friendly Python project or a fun coding exercise.
